* O que é o JavaScript (JS)?
* De onde surgiu?
* Qual o seu uso?
* Variáveis.
* Constantes.
* Tipos primitivos.
* Tipagem dinâmica.
* Operadores aritméticos.
* Operadores de atribuição.

* Criar códigos que usam estruturas condicionais, como o if/else.
